## Controls

| Button | Action |
|--|--| 
|DPAD|Move|
|Left Stick|Camera|
|A|Confirm|
|B|Cancel|
|X|Redo|
|Y|Show map|
|R1|Toggle for camera on dpad|
|L1|Cycle gutters/terrors|
|L2/R2|Zoom|
|Start|Menu|
|Select|Tooltip|
